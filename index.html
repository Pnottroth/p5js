Programmeren in p5.js

P5.js is een open-source JavaScript bibliotheek waarbij het mogelijk is om visuele media te creëren die makkelijk en toegankelijk is voor iedereen.

Basisprincipes van programmeren

Zoals we weten is taal een basisprincipe voor mensen om te communiceren. Woorden zelf zijn abstract, maar ze verwijzen naar een betekenis, naar een object of een actie. Programmeren is ook een taal dat wordt geschreven in de vorm van een code. Een code is een paragraaf met instructies of terwijl een to-do-lijstje. Dit to-do-lijstje moet met veel detail omschreven worden. Je schrijft tijdens het programmeren een code en probeert hiermee iets te vertellen wat het moet gaan doen. Bijvoorbeeld waar je bepaalde tekst wil hebben, vlakken, vormen, afbeeldingen etc. Het kan ook complexer je kan een code schrijven door iets te laten bewegen. Een code schrijven is eigenlijk een instructie die je mee geeft wat verandering te weeg brengt.

Onderdelen programmeren

Variabelen

Variabele is code dat wordt gebruikt om waardes zoals getallen op te slaan. Je kunt bijvoorbeeld een variabele hebben die de x-positie van een bal onthoudt, de kleur van een vierkant of de score van een spel. Door variabelen te gebruiken hoef je minder te schrijven en dat maakt een variabele heel handig. Bijvoorbeeld:
X = 15
X + X = 30
Je geeft de variabelen een logische naam bijvoorbeeld var studentenaam zodat je altijd kan nagaan waar de variabelen naartoe verwijst. Je hebt in p5.js build in variabelen die je kan vinden op de website.

Objects

Object is een verzameling van eigenschappen die alles bij elkaar samenvoegt. Dus we hebben een object bijvoorbeeld een student en daar komen allemaal eigenschappen in voor die samen het object vormen. Bijvoorbeeld:
• Leerjaar
• Schoolopleiding
• Geboortejaar
• Woonplaats
Alles in javascript is een object en kan opgeslagen worden in een variabelen. Hier links hebben we het over variabelen. Je kan bijvoorbeeld variabelen samenvoegen en er een object van maken.

Arrays

Met arrays voeg je meerdere waarde in een woord samen. Bijvoorbeeld:
var mijnStudenten = [‘Lisa’, ‘Jaap’, ‘Ashley, ‘Nick’].
Namen moeten in quotes geschreven worden. Als je wil verwijzen naar een studenten dan doe je dat zo:
mijnStudenten[0] (0 is het startpunt dus wordt verwezen naar Lisa)
mijnStudenten[2] (Is Ashley)
Voor de leesbaarheid van coderen is het handig om bijvoorbeeld de S van mijnStudenten kaplocks te maken zo valt het woord meer op.

LOOPS

Een for-loops herhaalt de input in een bepaalde volgorde. for-loops worden meestal gebruikt als er een herhaling moet worden uitgevoerd. Een for-loops zorgt ervoor dat de code kort blijft en niet dat je de code nog vijfitig keer onder elkaar hoeft te kopiëren en plakken.

FUNCTIONS

Bij p5.js heb je al twee standaard functies namelijk: setup function en draw function . Deze functies behoren tot het p5.js bibliotheek. Je kan ook je eigen functies maken en oproepen. Als je meerdere codes hebt geschreven die van alles en nog wat doen is het fijn om met functies het programma te organiseren. Dit kan je doen door de code onder te verdelen in stukken en die in verschillende functies zetten met een bijpassende functie naam.

IF/ELSE

Je schrijft in code stapsgewijs op wat het programma moet doen, maar wat als er verschillende antwoorden zijn op een vraag. Daardoor zijn er ‘if’-statements en de ‘else’-statements ontstaan. Er is niet altijd dat elke if-statement ook een else-statement heeft. Aan de andere kant hoort wel elke else-statement bij een if-statement. Stel je bent bezig met een code schrijven en je wil iets laten bewegen wat als de bal naar rechts gaat wat gebeurd er dan? Wat als de bal verder gaat dan het scherm wat gebeurd er dan? Je kan maar 1 if of else maken om toch er meer uit te halen kan je bijvoorbeeld met ‘else if’ code werken.

Voor wie is P5.js?

P5.js is toegankelijk voor ontwerpers, docenten, beginners, creatievelingen en ieder ander die voor het eerst code wil schrijven. P5.js is gratis en een open-source bibliotheek zodat het toegankelijk is voor iedereen om te gebruiken. Dus zeker aan te raden voor mensen die nog nooit eerder hebben gecodeerd. Ook heeft P5.js community ervoor gezorgd dat mensen die slechtziend zijn ook kunnen coderen.

Waarom P5.js?

Naast dat het gratis is en toegankelijk voor iedereen is het ook makkelijk voor een beginner, want coderen kan heel lastig en overweldigend zijn. P5.js biedt een web editor met een open-source bibliotheek aan die makkelijk te gebruiken is met hulp van het internet. Daarnaast hoef je ook geen software te downloaden, maar kan je makkelijk via de webbrowser je code schrijven. Het is wel handig als je een account hebt dan kan je onder je eigen naam al je codes opslaan.

Wat is p5.js?

Met p5.js kan je op een veilige plek het programmeren rustig onder de knie krijgen. p5.js is een bibliotheek aan functies met als taal JavaScript. Je kunt hier de beginstappen van programmeren leren. p5.js heeft een web editor waarin je je code kan schrijven. De web editor laat meteen resultaat zien van wat je codeert. Op de website van p5.js kan je functies terug vinden die p5.js heeft.
